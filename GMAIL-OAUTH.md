# Gmail OAuth2 Setup - Required

## إذا عندك OAuth2 Credentials:

### Google Cloud Console:
1. Go to: https://console.cloud.google.com
2. Create Project
3. Enable Gmail API
4. Credentials → OAuth Client ID
5. Get:
   - Client ID
   - Client Secret

### Then send me:
- Client ID
- Client Secret

---

## أو:

### Use App Password (easier):
1. https://myaccount.google.com/apppasswords
2. Create new app password
3. Send me the 16-digit password
