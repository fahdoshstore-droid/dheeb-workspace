# 🐺 Twitter/Telegram Signal Format

---

## 📝 Short Format (للإشارات السريعة):

```
🐺 ICT Signal
Bias: [Bearish/Bullish] (HTF aligned)
Liquidity: [External/Internal] [BSL/SSL] swept ([Session])
Entry: [price]
SL: [price]
TP: [price] / [price]
RRR: 1:[ratio]
Reason: [pattern] + [pattern] in [Premium/Discount]
```

---

## 💡 مثال:

```
🐺 ICT Signal
Bias: Bearish (HTF aligned)
Liquidity: External BSL swept (London)
Entry: 4215
SL: 4230
TP: 4180 / 4160
RRR: 1:3
Reason: MSS + Bearish FVG in Premium
```

---

## 📋 العناصر:

| العنصر | الوصف |
|--------|-------|
| Bias | HTF الإتجاه |
| Liquidity | نوع السيولة + الجلسة |
| Entry | سعر الدخول |
| SL | Stop Loss |
| TP | Take Profits |
| RRR | نسبة المخاطرة |
| Reason | السبب + المنطقة |
