## 📋 كيفية الاستخدام

### الخطوة 1: فتح صفقة
```
أنت: "فتح صفقة NQ شراء 21450 21420 21500"

الرد:
✅ صفقة مفتوحة:
- الاستراتيجية: SMC Scalping
- الدخول: 21450
- Stop Loss: 21420 (-30 نقطة)
- Take Profit: 21500 (+50 نقطة)
- المخاطرة: 1% من الرصيد

🎯 الالتزام بـ SL/TP الآن.
```

### الخطوة 2: إغلاق صفقة
```
أنت: "أغلق الصفقة عند 21500"

الرد:
✅ صفقة رابحة:
- النتيجة: +50 نقطة
- الوقت: 45 دقيقة
- PnL: +100 ريال

📊 الإحصائيات:
- إجمالي الصفقات: 5
- الفوزة: 3 (60%)
- Win Rate: 60%
- الربح الإجمالي: +250 ريال

🎯 نمط صحيح. استمر.
```

### الخطوة 3: طلب تقرير
```
أنت: "تقرير اليوم"

الرد:
📊 تقرير 2026-02-12:
- 3 صفقات مغلقة
- 2 فوز، 1 خسارة
- Win Rate: 66%
- PnL: +350 ريال

اليوم كان جيد. الالتزام يعمل.
```
