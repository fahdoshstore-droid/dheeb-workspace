# 📧 Gmail Setup - الحل النهائي

## المشكلة
Google blocked - يحتاج 2-Step Verification

## الحل بالتفصيل

### Step 1: شغّل 2-Step Verification
1. افتح: https://myaccount.google.com/security
2. ابحث عن "2-Step Verification"
3. شغّله (تفعيل)

### Step 2: أنشئ App Password
بعد ما تفعل 2-Step:
1. https://myaccount.google.com/apppasswords
2. App: Mail
3. Device: Other (DHEEB)
4. انسخ الـ 16 digit password
5. أرسله لـ DHEEB
