# 🐺 DHEEB Telegram Coach - دليل الاستخدام

## الملخص السريع
أنت معك مدرب فوري 24/7 بالـ Telegram. بدون تأخير. بدون كلام فارغ. فقط إشعارات تعطيك الإجابة اللي تحتاجها.

---

## **الخطوة 1: الإعدادات الأولية**

### ما تحتاج:
1. ✅ Notion account (استقبلت invitation)
2. ✅ Telegram account
3. ✅ Bot token (من BotFather)

### الإعداد:
```bash
# اذهب إلى المجلد
cd /home/ubuntu/.openclaw/workspace

# نشّغل النظام
node run-coach.js

# أو استخدم pm2 (يبقى يشتغل في الخلفية)
pm2 start run-coach.js --name "dheeb-coach"
pm2 logs dheeb-coach
```

## **الخطوة 2: الأوامر الأساسية**

### عند بدء التداول:
```
/start
```
- يعطيك ترحيب + القواعد الذهبية
- خيارات سريعة للأوامر الأساسية

---

### قبل دخول أي صفقة:
```
/check
```

**ما يحصل:**
- يسأل 7 أسئلة Pre-Trade Checklist:
  1. ✅ تأكيدات على الماكرو (4H + 1D)?
  2. ✅ نمط واضح (IFVG + OB + BOS)?
  3. ✅ الوقت صحيح (London/Asia/NY)?
  4. ✅ RRR = 1:2 بالضبط?
  5. ✅ احسبت 10 ثوان قبل الدخول?
  6. ✅ كام صفقة دخلت اليوم (≤2)?
  7. ✅ خسرت كام بالفعل (<$600)?

**اللي تفعل:**
- أجب عن كل سؤال بـ "نعم" أو "لا"
- لو كل الإجابات "نعم" → ادخل الصفقة
- لو في جواب واحد "لا" → ما تدخل

---

### لما تخلص الصفقة:
```
/analyze
```

**الصيغة الدقيقة:**
```
EURUSD | 1.0850 | 1.0860 | 1.0900 | 1.0830 | WIN | 5MIN | Pro
```

أو كل معلومة على سطر:
```
Symbol: EURUSD
Entry: 1.0850
Exit: 1.0860
TP: 1.0900
SL: 1.0830
Time: 5MIN
Result: WIN
Mistakes: none
```

**البيانات المطلوبة:**
- `SYMBOL`: الزوج (EURUSD, GBPUSD, إلخ)
- `ENTRY`: سعر الدخول
- `EXIT`: سعر الخروج
- `TP`: الـ Take Profit
- `SL`: الـ Stop Loss
- `RESULT`: WIN / LOSS / BREAKEVEN
- `PSYCHOLOGY`: Pro / Chaos / Greedy / Fear

**ما يحصل:**
- يحفظ الصفقة في Notion
- يحلل فوري:
  - من فاز (أي شخصية حاضرة)
  - وش الدرس
  - شنو الخطوة التالية

---

### عشان تتعرف على نفسك:
```
/personality
```

**الخيارات:**
- 🔴 Mr. Chaos - عاطفي، يقاتل السوق
- 🟠 Mr. Greedy - مستعجل، خايف يفوت الفرصة
- 🔵 Mr. Fear - خايف، محتاج تشجيع
- 🟢 Mr. Pro - منطقي، هادي، محترف

**ما يحصل:**
- يعطيك تحليل شخصي
- نصائح محددة للشخصية اللي أنت فيها

---

### عشان تشوف خطتك اليومية:
```
/plan
```

**ما يعطيك:**
- الجلسة الحالية (London / Asia / NY)
- أهدافك لليوم
- الأوقات المسموح فيها
- النماط اللي تبحث عنها
- القواعس قبل وبعد الصفقة

---

### عشان تشوف إحصائياتك:
```
/stats
```

**ما يعطيك:**
- إجمالي الصفقات (آخر 7 أيام)
- Win Rate
- Profit Factor
- Discipline Score
- أكثر خطأ متكرر
- Trend الحالي (صاعد / هابط / ثابت)

## **الخطوة 11: Troubleshooting**

### مشكلة: "Bot ما يرد"
```bash
# تحقق من الـ token
echo $TELEGRAM_BOT_TOKEN

# تحقق من الـ chat id
echo $TELEGRAM_CHAT_ID

# أعد تشغيل
pm2 restart dheeb-coach
pm2 logs dheeb-coach
```

### مشكلة: "Notion ما يحفظ البيانات"
```bash
# تحقق من الـ key
echo $NOTION_API_KEY

# تحقق من الـ access
curl -X GET "https://api.notion.com/v1/users/me" \
  -H "Authorization: Bearer $NOTION_API_KEY"
```

### مشكلة: "Bot يكرر نفس الرسالة"
```bash
# اقتل وأعد تشغيل
pm2 delete dheeb-coach
pm2 start run-coach.js --name "dheeb-coach"
```
