# Perplexica - AI Search Engine

## Status
- **Running**: ✅ Yes
- **URL**: http://localhost:3000
- **Port**: 3000

## What is Perplexica?
AI-powered answering engine that runs entirely on your own hardware.

## Features Useful for Us

| Feature | Project Benefit |
|---------|---------------|
| AI Search | Find trading setups |
| Local LLM (Ollama) | Free AI without API costs |
| SearxNG | Private web search |
| Research | Find project info |

## How to Use

1. Open http://localhost:3000
2. Configure AI provider (Ollama or OpenAI)
3. Start searching

## Docker Commands

```bash
# Start
docker start perplexica

# Stop
docker stop perplexica

# Logs
docker logs perplexica

# Restart
docker restart perplexica
```
