# ICT/SMC - Full Guide

## 1. ORDER BLOCKS (OB)

### What is it?
Candle(s) before a strong impulse move.

### Types

| Type | Pattern | Trade |
|------|---------|-------|
| Bull OB | Green → Strong Up | Long |
| Bear OB | Red → Strong Down | Short |

### How to Find
1. Find strong move (100+ pts)
2. Go back 1-3 candles
3. That's your OB

### How to Trade
- Wait for price to return to OB
- Enter at OB
- SL below/above OB

---

## 2. FAIR VALUE GAP (FVG)

### What is it?
Gap between 3 consecutive candles.

### Bull FVG
```
Candle 1: Close > Open (Green)
Candle 2: Low < Low[1]
Candle 3: Low < Low[1]
```
Price creates upward gap → Bullish

### Bear FVG
```
Candle 1: Close < Open (Red)
Candle 2: High > High[1]
Candle 3: High > High[1]
```
Price creates downward gap → Bearish

### How to Trade
1. Find FVG on chart
2. Enter when price returns to fill FVG
3. SL below/above FVG

## 3. LIQUIDITY

### Levels

| Level | Meaning | Symbol |
|-------|---------|--------|
| PDH | Previous Day High | BSL |
| PDL | Previous Day Low | SSL |
| PWH | Previous Week High | BSL |
| PWL | Previous Week Low | SSL |
| PMH | Previous Month High | BSL |
| PML | Previous Month Low | SSL |

### Liquidity Sweep
```
BSL Sweep: Price goes up to take liquidity → then drops = Bearish
SSL Sweep: Price goes down to take liquidity → then rises = Bullish
```

## 6. TRIL FRAMEWORK

### T - Trend
- HTF Bias: Bullish/Bearish/Neutral
- BoS/CHoCH: Continuation or reversal
- HH/LL: Higher Highs/Lower Lows

### R - Raid (Liquidity)
- BSL: Buy Side Liquidity (Above)
- SSL: Sell Side Liquidity (Below)
- Sweep: Liquidity grab

### I - Imbalance
- FVG: Fair Value Gap
- OB: Order Block
- MSS: Market Structure Shift

### L - Location
- Kill Zone: Session timing
- Premium/Discount
- Session: Current session

## 7. OTE - Optimal Trade Entry

### How to Use

2. Find impulse move
