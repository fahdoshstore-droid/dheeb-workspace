# 🧠 Mental Guard - الحارس النفسي

## الهيكل الجديد:

```
Behavioral Guard (السلوك)
    ↓
Risk Guard (المخاطر)
    ↓
ICT Guard (تحليل السوق)
    ↓
Execution (التنفيذ)
```

---

## 1. Behavioral Guard 🧠

**قبل أي شيء:**

```
→ Am I angry? → STOP
→ Am I revenge trading? → STOP
→ Am I tired? → STOP
→ Am I greedy? → STOP
→ Did I lose 3 in a row? → STOP 30 min
```

---

## 2. Risk Guard 💰

**قبل الدخول:**

```
→ Risk ≤ 2%? → Yes/No
→ R:R ≥ 2.5? → Yes/No
→ Daily loss < $600? → Yes/No
→ Max 2 trades done? → Yes/No
```

---

## 3. ICT Guard 📊

**التحليل:**

```
→ Killzone? (NY 9:30-11:30 UTC)
→ Bias clear? (D/4H)
→ Liquidity sweep? 
→ FVG/OB formed?
→ SMT confirmed?
→ Entry trigger?
```

---

## 4. Execution 🎯

**إذا اجتزت الـ 3 guards:**

```
→ Execute trade
→ Place SL
→ Place TP1/TP2/TP3
→ Log trade
```

---

## ملخص Flow:

```
ANY TRADE
    ↓
🧠 Behavioral Guard
    ↓ Pass? ↓
💰 Risk Guard
    ↓ Pass? ↓
📊 ICT Guard
    ↓ Pass? ↓
🎯 Execution
    ↓
📝 Log
```

---

## القواعد الذهبية:

```
❌ NO TRADE without passing ALL 4 guards
❌ NO override (even if "sure")
❌ NO skip steps
```
