# Trade Flow Protocol - ICT + Pro Traders

## قبل كل صفقة (Pre-Trade)

### 1. HTF Analysis (1H/4H)
- [ ] حدد الـ Bias (Bullish/Bearish/Range)
- [ ] ابحث عن Market Structure Shift
- [ ] حدد الـ PD Arrays

### 2.Confluences (3+ required)
- [ ] Liquidity Sweep occurred?
- [ ] Order Block / FVG formed?
- [ ] Kill Zone timing?
- [ ] MSS on lower TF?
- [ ] OTE in discount/premium?

### 3. Risk Calculation
- [ ] Risk % = 0.5-1% max
- [ ] SL level محدد
- [ ] TP level محدد
- [ ] R:R ≥ 1:1.5

### 4. Psychology Check
- [ ] مش في حالة ممنوعة (revenge/fomo/anger)?
- [ ] ناوي تدخل on plan مو on emotion?

---

## أثناء الصفقة (During Trade)

### 1. Execution
- [ ] Entry من zone صحيحة
- [ ] SL تحت/فوق الـ Invalidation

### 2. Management
- [ ] Move to breakeven عند 1:1
- [ ] Partial at 1:2
- [ ] Trail with structure

### 3. Monitoring
- [ ] ما تعدل الاستوب للخسارة
- [ ] ما تدخل صفقة ثانية إذا واحدة شغالة
- [ ] لاحظ أي تغيير في الـ narrative

---

## بعد الصفقة (Post-Trade)

### 1. Journal
- [ ] Screenshot
- [ ] Entry reason
- [ ] Exit reason
- [ ] Emotional state
- [ ] R Multiple

### 2. Review
- [ ] Win or Loss - accept it
- [ ] What went right?
- [ ] What went wrong?
- [ ] Lesson learned?

### 3. Psychology Reset
- [ ] امسح أي emotion
- [ ] استعد للجلسة التالية
- [ ] ما تفكر في الصفقة السابقة
