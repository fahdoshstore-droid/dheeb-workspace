## 🎯 Core Strategy (A+ Model Optimized)

### 1. Top-Down Analysis (MANDATORY)
```
Weekly → Daily → 4H → 1H → 15m → 5m
```
- Weekly: Trend direction + PDR (Previous Day Range)
- Daily: PDR + PWH (Previous Week High) + PWL
- 1H/4H: Order Blocks + Fair Value Gaps
- 15m/5m: Entry execution

### 2. Entry Criteria (The Confluence Stack)
✅ Required for EVERY trade:
1. **Liquidity Sweep** - High/Low of previous session swept
2. **FVG / IFVG** - Fresh Fair Value Gap formed
3. **MSS** - Market Structure Shift confirmed
4. **Order Block / Breaker** - Smart money entry zone
5. **SMT** - Smart Money Tool divergence (optional but preferred)

### 3. Time-Based Entries
- **London Session**: Best for SD1-SD2 (Support/Demand 1-2) rejections
- **NY Open**: Wait 30-45 min after open (liquidity grab)
- **Avoid**: News time, Friday after 2pm EST

## ⚡ Quick Reference - Entry Signals

### Long Entry
1. Price sweeps recent low
2. Forms FVG (bullish gap)
3. Returns to fill FVG
4. Forms Order Block
5. Enter on OB touch or FVG fill
6. SL below OB

### Short Entry
1. Price sweeps recent high
2. Forms FVG (bearish gap)
3. Returns to fill FVG
4. Forms Breaker Block
5. Enter on breaker touch
6. SL above breaker
