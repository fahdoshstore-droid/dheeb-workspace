## 🔄 Decision Flow (من الصور)

```
1. News?
   ├── Yes → WAIT
   └── No → Continue
   
2. HTF Bias (D/4H)
   ├── Clear → With Bias Path
   └── Unclear → Neutral Path
   
3. With Bias Path:
   → Premium/Discount Levels
   → HTF PDAs (OB/FVG/Breaker) on H1/M15
   → M1/M5 Entry Trigger
   → Execute if MMXM aligned
   
4. Neutral Path:
   → Soft Bias (RTH Gap / NWOG)
   → REH/REL targets
   → Entry at extremes
```

## 🎯 Entry Types

### A. With Clear Bias (Trending)
1. Price reaches HTF key level
2. Forms OB/FVG on H1/M15
3. Wait for M1/M5 trigger
4. Entry at OB touch or IFVG fill
5. SL: below recent liquidity
6. TP: PDH/PDL or SSL/BSL

### B. Neutral (Rangebound)
1. Wait for price at extremes
2. Premium ( Sell) / Discount (Buy)
3. Look for REH/REL
4. Entry with soft bias
5. TP: Range equilibrium

### C. All-Time High
1. No historical levels
2. Use Standard Deviations (SD1, SD2, SD3)
3. Wait for rejection from SD level
4. Entry on FVG/OB

## 📝 Examples

### Long Setup (ICT):
1. Bearish market (HTF down)
2. Price sweeps recent low (liquidity)
3. Forms bullish FVG on H1
4. Returns to fill FVG
5. Forms Order Block
6. Enter on OB touch
7. SL below liquidity
8. TP: PDH or previous high

### Short Setup (ICT):
1. Bullish market (HTF up)
2. Price sweeps recent high
3. Forms bearish FVG
4. Returns to fill
5. Enter at FVG/OB
6. SL above
7. TP: PDL or SSL
