## 📈 Strategy: ICT/SMC

### Timeframes
Weekly → Daily → 4H → 1H → 15m → 5m → 1m

### Sessions
- **London Kill Zone:** 8:00-11:00 UTC ✅
- **NY Kill Zone:** 9:30-11:30 UTC ✅ (Primary)

### Entry Setup (Long)
1. Price sweeps recent **Low** (Liquidity Grab)
2. Forms **FVG** (Bullish gap up)
3. Returns to fill FVG
4. Forms **Order Block** (Bear candle → Bull candle)
5. Enter on OB touch or FVG fill
6. SL below recent liquidity

### Entry Setup (Short)
1. Price sweeps recent **High**
2. Forms **FVG** (Bearish gap down)
3. Returns to fill FVG
4. Forms **Order Block**
5. Enter on OB touch or FVG fill
6. SL above recent liquidity
