# 📋 Daily Trading Checklist

## قبل أي صفقة

- [ ] **الوقت مناسب؟** (NY Killzone: 9:30-11:30 UTC)
- [ ] **Bias واضح؟** (D/4Htrend)
- [ ] **المستويات محددة؟** (PDH, PDL, Weekly)
- [ ] **Liquidity Sweep حدث؟** (Lows/Highs محرومين)
- [ ] **Setup موجود؟** (OB + FVG)
- [ ] **R:R ≥ 2.5؟**
- [ ] **SL محدد؟** (منطقي)
- [ ] **حجم العقد ضمن الحد؟** (Max 2%)
- [ ] **حالة نفسية OK؟** (لا revenge، لا fear)
