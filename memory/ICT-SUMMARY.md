# ICT Summary - Quick Reference

## Core Concepts

### Market Structure
- **Trend:** Higher highs + higher lows (bullish)
- **Trend:** Lower highs + lower lows (bearish)
- **Structure Break (BoS):** Price breaks previous structure
- **CHoCH:** Change of Character - trend change

### Key Levels
- **BSL:** Buy Side Liquidity (buyers waiting)
- **SSL:** Sell Side Liquidity (sellers waiting)
- **OB:** Order Block - institutional order zone
- **FVG:** Fair Value Gap - inefficiency
- **MB:** Mitigation Block
- **BB:** Breaker Block

### ICT Terms
- **DOL:** Draw on Liquidity
- **SMT:** Smart Money Technique (divergence)
- **POI:** Point of Interest
- **LTF/HTF:** Low/High Timeframe

---

## Trading Rules

1. **No News** - Check calendar first
2. **Bias** - From Weekly/Daily
3. **Wait for Macro** - Specific time windows
4. **Liquidity Sweep** - Must happen first
5. **FVG + CHoCH** - Entry confirmation
6. **R:R ≥ 2.5** - Minimum
7. **Stop Loss** - Always required
8. **Max 2 trades/day**
9. **Max -2% daily loss**

---

## Macros (Best Times)

- **London:** 02:33-03:00, 04:03-04:30
- **NY AM:** 08:50-09:10, 09:50-10:10, 10:50-11:10
- **Killzone:** 09:30-11:30
