# تحليل COT والسندات لتحديد الاتجاهات

## تحليل COT

يكشف عن مواقف المؤسسات الكبرى (شراء/بيع)

- **Long ↑** = اتجاه Bullish (صاعد)
- **Short ↑** = اتجاه Bearish (هابط)

---

## أماكن الانعكاس

الجمع بين:
- بيانات COT
- السندات
- البلوكات (OB/BB)
- Fair Value Gaps

= تحديد الانعكاسات بدقة

---

## السندات

- **عوائد ↑** = احتمال صعود العملة (Bullish)
- **عوائد ↓** = احتمال هبوط العملة (Bearish)

---

## الخلاصة

COT + Bonds + Chart Levels = توضيح الاتجاه وتوقع الانعكاسات
