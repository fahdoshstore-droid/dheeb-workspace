# 📋 Daily Trading Checklist v3

```
1️⃣ News?
   → نعم؟ → WAIT

2️⃣ Bias (D/4H)?
   → Clear؟ → Bias Path

3️⃣ Sweep؟

4️⃣ Setup?

   → Divergence confirmed؟

6️⃣ Entry Trigger?
   → M1/M5 IFVG/Breaker؟

7️⃣ R:R ≥ 2.5؟

8️⃣ SL placed؟

9️⃣ Size ≤ 2%؟

🔟 psyche OK؟
```
