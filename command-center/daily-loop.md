# Daily Loop

## Morning (Start of Day)
1. Check priorities.json
2. Review current trades
3. Check Kill Zone times
4. Update state

## During Kill Zone
1. Monitor for setups
2. If setup exists → Analyze
3. Get approval from Fahad
4. Execute
5. Log immediately

## End of Day
1. Log today's trades
2. Calculate P/L
3. Update stats
4. Answer: Did I follow the system?
5. What mistakes happened?
6. What improvement?

## Night
- No trading
- System improvements
- Business tasks
