## 🎯 Entry Setups (A++ + IFVG)

### Setup #1: London Session
```
1. Wait for protection to SD1-SD2
2. Price retraces with displacement
3. Form FVG/OB/IFVG
4. Entry on retest
5. Target: Previous day/week high
```

### Setup #2: NY Session (IFVG Model)
```
1. Previous day low/high sweep
2. IFVG forms (after displacement)
3. Entry on IFVG confirmation
4. Target: Previous wick/data
5. Works best on NQ during NY
```

### Setup #3: Standard ICT
```
1. Weekly Base (OB/FVG/Breaker)
2. Daily Fractal Shift
3. Internal confirmation
4. FVG/OB + retest
5. Target: Liquidity
```
