## قبل أي صفقة (Pre-Trade Gate)

**كل شارت → أسأل فوراً:**

1. **"Entered?"** ← هل دخلت؟
2. **"State 1-10?"** ← حالتك النفسية؟
3. **"Detached?" (Y/N)** ← متعلق بالنتيجة؟

**إذا:**
- State < 8 → STAND DOWN
- Answer ≠ Y → STAND DOWN
